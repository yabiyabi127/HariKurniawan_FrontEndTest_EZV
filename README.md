# Todo List Apps

Ini adalah Tes dari PT. EZV Service Indonesia yaitu aplikasi sederhana Todo List Apps dari API dari PT. EZV Service Indonesia dengan memanfaatkan teknologi Nextjs, tailwind CSS, typescript, Eslint dan redux-tool-kit, redux-tool-kit-query

## 🚀 Fitur Implementasi

- ✅ Get Api from https://jsonplaceholder.typicode.com
- ✅ Add Data Todo (POST) Api from https://jsonplaceholder.typicode.com
- ✅ Consume API
- ✅ Implement redux-tool-kit-query (Store management state)
- ✅ Implement Server Side Rendering (SSR)
- ✅ Implement Incremental Static Regeneration (ISR)

## 🛠️ Teknologi

- [Nextjs](https://nextjs.org/docs)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [ESLint](https://eslint.org/)
- [Redux-tool-kit](https://redux-toolkit.js.org/)

## 🧰 Instalasi

```bash
git clone https://github.com/yabiyabi127/HariKurniawan_FrontEndTest_EZV.git
cd todo-apps-nextjs
npm install
npm run dev
```
